Algorithm Visualizer

2020

Sorting algorithms were always difficult for me to understand. I created this tool to to help me better grasp their nature by visualing the process. This project definitely gave me more confidence with understanding algorithms.

Space Shooter

2019

I've always had a fascination with game development, so I set out to create a small game that eventually became a standalone web-supported demo. The world of game engines offered several choices, including Unreal, Godot, LibGDX, and Solar2D. In the end, I chose Unity because of its well-documented features, it's support for web-based builds, and the abundance of online learning resources.

Species Study

2019

After studying endangered primates in my anthropology class, I decided to create a webpage dedicated to one special species: the golden snub-nosed monkey. This project enhanced my CSS skills and improved the presentation of websites. Specifically, I learned about parallax scrolling and bootstrap 4. I also learned how to use the Sheridan College domain to host webpages. Did you know these monkeys are known to engage in snowball fights? They form and throw snowballs just like humans.
